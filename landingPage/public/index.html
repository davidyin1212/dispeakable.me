Installation Guide

1. Download the extension file from the website and save it to your computer.

2. Click the hotdog icon to bring options menu on the browser toolbar.

(if you have an old version of Chrome, the icon is a wrench)

3. Select Tools > Extensions.

4. Locate the extension file on your computer and drag the file onto the Extensions page.

5. Review the list of permissions in the dialog that appears. If you would like to proceed, click Install.

6. Plugin icon should appear. Enjoy encrypting!

Sharing

All there needs to be done is to have the public key shared to the recipients, e.g. friends, is to set any secret key and hit "Generate," and another encrypted keypass will be printed below. Copy that code and send to your recipients.


Once your friend receives the passkey, paste it in the textarea above "Add Friend," and enter in the Secret key. He/she will be permanently added to your listings which will be used in the future in encrypt any messages intended for that specific friend.
